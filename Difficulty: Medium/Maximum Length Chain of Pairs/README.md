<h2><a href="https://www.geeksforgeeks.org/problems/maximum-length-chain-of-pairs/1">Maximum Length Chain of Pairs</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p data-start="469" data-end="745"><span style="font-size: 14pt;">You are given an array <strong>arr[]</strong> of n pairs where each pair consists of two integers <code data-start="548" data-end="556">(a, b)</code> such that <code data-start="567" data-end="574"><strong>a &lt; b</strong></code>.</span><br data-start="575" data-end="578"><span style="font-size: 14pt;"> A pair <code data-start="585" data-end="593">(c, d)</code> can follow another pair <code data-start="618" data-end="626">(a, b)</code> if and only if<strong data-start="627" data-end="645"> <code data-start="646" data-end="653">b &lt; c</code></strong>.</span><br data-start="654" data-end="657"><span style="font-size: 14pt;"> A <strong data-start="659" data-end="677">chain of pairs</strong> is a sequence of pairs where each pair can follow the previous one.</span></p>
<p><span style="font-size: 14pt;"> </span></p>
<p data-start="747" data-end="873"><span style="font-size: 14pt;">Return the <strong data-start="758" data-end="789">length of the longest chain</strong> that can be formed using the given pairs.</span><br data-start="831" data-end="834"><span style="font-size: 14pt;"> You may arrange the pairs in any order.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr[][] = [[5, 24], [15, 25], [27, 40], [50, 60]]
<strong>Output: </strong>3
<strong>Explanation: </strong>One of the longest pairs is [5, 24] -&gt; [27, 40] -&gt; [50, 60] .</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr[][] = [[6, 8], [3, 4]]
<strong>Output: </strong>2
<strong>Explanation: </strong>The longest chain pair is [3, 4] -&gt; [6, 8]</span></pre>
